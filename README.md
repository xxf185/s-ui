


## 默认安装信息
- 面板端口：2095
- 面板路径：/app/
- 订阅端口：2096
- 订阅路径：/sub/
- 用户/密码：admin

## 安装并升级到最新版本

```sh
bash <(curl -Ls https://raw.githubusercontent.com/xxf185/s-ui/master/install.sh)
```

## 安装自定义版本

安装自定义版本.请将版本添加到安装命令的末尾.例如，版本“0.0.1”：

```sh
bash <(curl -Ls https://raw.githubusercontent.com/xxf185/s-ui/master/install.sh) 0.0.1
```
